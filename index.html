Cómo conectar Firebase

Guía paso a paso para que la base de datos esté en la nube y accesible desde cualquier dispositivo.

🔥 ¿Qué es Firebase Firestore?

Firebase es un servicio gratuito de Google que te permite guardar datos en la nube. Con Firestore (su base de datos), los datos están disponibles desde cualquier computadora o celular, en tiempo real. No necesitás configurar ningún servidor.

✅ Gratis hasta ciertos límites ☁️ Nube de Google ⚡ Tiempo Real
📋 Pasos para Configurar
1
Crear una cuenta en Google / Firebase
Entrá a https://console.firebase.google.com y hacé clic en "Agregar proyecto". Dale un nombre como "gestor-alumnos". Desactivá Google Analytics (no lo necesitamos) y hacé clic en "Crear proyecto".
2
Crear la base de datos Firestore
En el menú lateral izquierdo buscá "Firestore Database" y hacé clic en "Crear base de datos". Elegí "Modo de prueba" (permite leer y escribir sin restricciones por 30 días, perfecto para empezar). Elegí la región "us-central" (la más cercana disponible para Argentina).
3
Registrar tu app Web
En "Configuración del proyecto" (ícono de engranaje ⚙️ arriba a la izquierda), bajá hasta "Tus apps" y hacé clic en el ícono </> (Web). Dale un nombre a la app (por ej: "gestor-web") y hacé clic en "Registrar app". Firebase te mostrará un objeto firebaseConfig con tus claves.
4
Pegar la configuración en el código
Abrí el archivo index.html con cualquier editor de texto (Bloc de Notas, VS Code). Buscá el comentario que dice "REEMPLAZÁ ESTOS VALORES" y reemplazá el objeto firebaseConfig con el que te dio Firebase:
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "mi-proyecto.firebaseapp.com",
  projectId: "mi-proyecto-id",
  storageBucket: "mi-proyecto.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456:web:abc123"
};
5
Configurar la clave de Anthropic (para la IA)
Para usar la búsqueda por IA y la extracción de PDFs, necesitás una clave de API de Anthropic. Entrá a https://console.anthropic.com, creá una cuenta y generá una API key. Luego buscá en el código la línea con ANTHROPIC_API_KEY y pegala. ⚠️ Importante: esta clave NUNCA debe compartirse públicamente. En producción, usá un servidor backend para protegerla.
const ANTHROPIC_API_KEY = "sk-ant-api03-...";
6
Instalar la PWA en el celular
Para que la app funcione como una app instalada en el celular, subí los tres archivos (index.html, manifest.json, sw.js) a un hosting gratuito como Netlify (netlify.com) o Vercel (vercel.com). Después, desde el celular, abrí la URL en Chrome y elegí "Agregar a pantalla de inicio".
7
Reglas de seguridad en Firestore
Para proteger los datos (que solo personas autorizadas puedan leer/escribir), en la consola de Firebase → Firestore → "Reglas", pegá esto para que solo funcione desde tu dominio:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /alumnos/{doc} {
      allow read, write: if true; // Cambiá por autenticación en producción
    }
  }
}
💡 ¿Qué pasa si no configurás Firebase?

No te preocupes. La app funciona perfectamente sin Firebase usando localStorage (el almacenamiento del navegador). La diferencia es que los datos solo estarán en esa computadora y ese navegador. Si limpiás el historial del navegador, los datos se borran. Firebase los guarda de forma permanente y accesibles desde cualquier lugar.

✅ localStorage: funciona sin internet ni configuración ☁️ Firebase: datos en la nube, acceso desde cualquier dispositivo
